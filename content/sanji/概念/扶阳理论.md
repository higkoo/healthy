# 扶阳理论

> **类型**: concept
> **创建时间**: 2026-07-21
> **最后更新**: 2026-07-21
> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)、[再论疫苗-倪海厦.md](../原始资料/再论疫苗-倪海厦.md)

## 摘要

扶阳理论是倪海厦医学体系的核心思想，源自《黄帝内经》和《伤寒论》，强调「阳主阴从」——阳气是人体生命活动的根本动力。倪师将其系统化为「倪氏六大健康准则」（正常人的生理标准）和「经方治病七法」，并发展了完整的诊断体系（望/问/脉/触/眼诊/辨阴阳），核心法则包括：法象天道（心为太阳）、阳不入阴（肿瘤成因）、治癌五法（断瘤之源/以阳制阴/疏导攻坚/活血化瘀/常顾胃气）、药简力专敢用毒药。扶阳派善用麻桂姜附硫磺等阳药，以四逆汤为急救代表方。

## 详情

### 一、知常达变——正常人的七条标准

> 「大道至简，老师本从于内经的论述制定了正常人的标准。」

1. 一年四季一日四时额凉手足温（体内能量运行正常，阴阳平衡之象）
2. 胃口好，饮食能自制（胃气平），食之有味（脾气充）
3. 睡觉容易入眠、能通宵（心肝血足），汗出正常（心气衡），睡时体表冬暖夏凉（表阳固密）
4. 大便每日晨起一行（肠道功能好）
5. 小便淡黄一天5-7次、睡时不小便（肾气旺）
6. 白天精神好，体力足（肺气足）
7. 脉一息四、五至和缓有力，春弦夏洪秋浮冬沉，寸脉大于尺脉，男子右脉大于左脉，女子反之（阴平阳秘）

> 「凡此七条，过与不及皆为病象。」

**知常达变的意义**：
- 先掌握正常人标准，才能识别异常
- 西医被细枝末节蒙蔽，忘了「何为正常」
- 天下的疾病千变万化，中医之妙就在于执简驭繁——明白了人的正常之象，异常之处也就了了分明

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)，第20-35行

### 二、精确诊断六法

倪师建立了六层诊断体系，目标是「用最短的时间、最简单的方法，获得最有效的信息，分清阴阳」：

#### 2.1 望法
「上工治病，望而知之」。倪师的望诊结合周易之学，核心是「形神气血，五行定位，色位生克，时色推移」。

#### 2.2 问法
「十问」是常法，倪师临诊细解后学生方知其中奥妙。

#### 2.3 脉法
化繁为简：识「四时平人之脉」，辩过与不及，知浮沉迟数、明中空结代，晓附骨阴实之象。

#### 2.4 触法
- **以手测皮温**：皮表寒热变化是体内阴阳变化的重要外征。额及手背应凉，手掌当热，重按手掌至骨可测骨之寒热
- **脊椎骨/背俞/经穴探察**：可迅速定位结实、肿瘤等疾病
- **腹诊**：集合历代经方腹诊之大成，直接指导经方的辨证使用
- **耳诊**：耳穴图是中华瑰宝，融诊治于双耳

#### 2.5 眼诊法
倪氏自创独门绝学，短短几秒就能从小小眼睛中看出五脏虚实，确定病之所在、邪之性。与四诊互相印证。

#### 2.6 辨阴阳法
中医诊断的最高境界。只有深刻领会内经的阴阳四时变化道理、气血循环的周天法度，才能真正掌握。

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)，第36-49行

### 三、法象天道——心为太阳的能量循环模型

倪师将人体比作天地系统：

```
太阳（心）→ 能量经天空（肺）中的水气下降 → 传递到大地四方（脾）
→ 渗透溪河（三焦）→ 灌溉大地滋养万物（肝）
→ 海（肾）中的水蒸发回到天空（肺）→ 循环不已
```

- **能源中心**：心（太阳）
- **能量循环媒介**：水
- **能量之源**：肠胃

> 「常态下，阳光灿烂，水道通畅，没有积水积湿，没有阴蔽之所，动物很难生存。」

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)，第50-58行

### 四、经方治病七法

倪师将经方治病思路归纳为七条法则：

| 法则 | 内容 | 常用药物 |
| --- | --- | --- |
| **1. 首重阳气，重视护心** | 心主神、主火，是人体所有阳气的源头。善用麻桂姜附硫磺等阳药 | 麻黄、桂枝、生姜、附子、硫磺 |
| **2. 阴阳谐调，谨护肠胃** | 桂芍等量一散一收；虚阳上浮用龙牡镇；表阳不固用炮附收；参姜枣草补脾生津 | 桂枝、白芍、龙骨、牡蛎、炮附子、人参、生姜、大枣、炙甘草 |
| **3. 上开玄府，下利水道** | 病在表以麻桂葛根行阳气从汗解；病在里以苓泽术夏甘淡渗湿从小便解 | 麻黄、桂枝、葛根、茯苓、泽泻、白术、半夏 |
| **4. 推陈致新，谷道常通** | 病在腑以硝黄枳朴通便去实 | 芒硝、大黄、枳实、厚朴 |
| **5. 调和枢机，疏理三焦** | 病在三焦水道以柴胡剂和解少阳 | 柴胡 |
| **6. 祛瘀陈挫，血脉通利** | 瘀血内停以水蛭芒虫桃仁白芍茜草活血化瘀 | 水蛭、芒虫、桃仁、白芍、茜草 |
| **7. 寒热兼施，补不忘泻** | 肠胃厥阴寒热并结时用桂附吴萸+芩连石膏；山药熟地山萸+茯苓丹皮泽泻一补一泻 | 桂枝、附子、吴茱萸+黄芩、黄连、石膏；山药、熟地、山茱萸+茯苓、丹皮、泽泻 |

**核心原则**：
- 最重视救护阳气——阳就是动能，是生命力
- 其次重视治水治瘀——保持水道、血脉和谷道通畅
- 能量和物质循环的主要路径通畅了，小的堵塞停积无碍整体

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)，第59-68行

### 五、阳不入阴——肿瘤成因与治癌五法

#### 5.1 肿瘤成因

> 「当身体内局部温度过低动能不足时，造成大量的营养物质和冷水停积聚集起来，局部压力逐渐升高，正常的津液和热能无力灌注和通过，于是阴实越积越重，肿瘤也越来越大。」

所有肿瘤（骨瘤、脑瘤、乳癌、大肠癌）共同点：**必有阳不入阴的表征，即所谓阴实已成**。

#### 5.2 治癌五法

1. **断瘤之源**：停吃一切人工制造的高浓度高营养食品（包括奶制品），断绝肿瘤的食物来源
2. **以阳制阴**：阳药使机体天然抗病力恢复——如红日在天，阴云自然四散无影
3. **疏导攻坚**：甘淡渗利+咸以软坚，把积存的营养攻散、冷水排除，降低病灶压力
4. **活血化瘀**：久郁必瘀，久积必瘀，久病必瘀，去瘀则生新
5. **常顾胃气**：脾胃为后天生化之本，胃气不绝必无性命之忧

#### 5.3 治癌时机

> 「每每先护心气，壮心肾之阳，强脾胃，守而待天时。一旦到了病脏的王时，速发重剂猛攻，时机一过，即刻收兵。如此反复拉锯，步步为营，且战且进。」

- 心王夏（立夏治心），肺王秋，肝王春，肾王冬
- 快者月余，慢者数月多获奇效

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)，第70-90行

### 六、药简力专，敢用毒药

> 「处方用药，一定要谨守经方法度。经方者经典之方，历经二千余年的印证，疗效确凿可靠，是最强的药了。只要对证，必能一剂知，二剂已。」

> 「如果病人吃了经方而无效，绝不是经方的错，一定是你辨证不准，或是药力未到。」

**治重病危候的原则**：
- 越重越险的病人，药反而越简单
- 敢用附子、细辛、硫磺、大黄、甘遂、大蓟等毒药峻剂
- 急攻存正，生死立判，绝不能手软
- 剂量不到药力不足，就算处方对了也救不回来

**临床案例**：
- 90岁老妪脑瘤重症（右眼已瞎，头痛如劈，脉洪数）→大承气汤加味，一剂中病
- 多发性骨髓瘤全身移转（四肢冰冷至肘膝，汗出不止，呼吸急促，骨痛难忍）→大剂四逆汤，危候暂退

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)，第84-90行

### 七、扶阳派与温病派的根本分歧

| 方面 | 扶阳派（倪师） | 温病派 |
| --- | --- | --- |
| 核心理论 | 阳主阴从，扶阳为本 | 滋阴清热 |
| 代表方 | 四逆汤、麻黄汤、大青龙汤 | 银翘散、桑菊饮 |
| 用药特点 | 善用麻桂姜附硫磺等阳药，敢用毒药峻剂 | 少用峻药，讲究温和滋补 |
| 治感冒 | 50元台币一剂第二天好 | 一两个月病人自愈 |
| 对叶天士评价 | 叶天士其实用很多麻桂附子，未脱离经方，近代温病学者只学到皮毛 | 视叶天士为温病派创始人 |

倪师观点：温病派认为南方湿热只有温病不用伤寒，完全错误。张仲景太阳病三方（中风桂枝汤、伤寒麻黄汤、温病葛根汤）——经方早就涵盖温病。

> **来源**: [传统中医跟诊过程-倪海厦.md](../原始资料/传统中医跟诊过程-倪海厦.md)、[梁冬对话倪海厦.md](../原始资料/梁冬对话倪海厦.md)

### 八、学生跟诊的三心

倪师学生总结跟诊最大收获是「三颗心」：
1. **信心**：看到真正中医的玄妙之处，对经方信心倍增
2. **慧心**：明白中医该如何下手去学，医书如山有路可循
3. **决心**：正统中医针灸和经方是上古大圣贤留下的神兵利器，有幸学到就要发扬它

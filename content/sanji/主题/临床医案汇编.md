## 详情

### 一、诊断框架

#### 十问（倪师问诊核心）

1. **睡眠**——失眠/易醒时段：1–3 点醒为肝有阴实，3–5 点醒为肺有阴实（凌晨 1~3 点气行肝经，3~5 点气行肺经；正常人肝肺无阴积可睡通宵）
2. **胃口**——过大为热（石膏证），差为寒（干姜证）；脏病夜间饿、腑病日中饿为将愈
3. **大便**——每日晨起一行为常（最正常者应卯时一行成条不稀）；便秘不痛为寒实，痛为热实；肝病必治大肠，下利比便秘好
4. **小便**——淡黄为常；深黄为热/有宿便；清白为寒；小便色可侦测肠胃宿便与燥热（膀胱与小肠紧邻）
5. **口渴**——喜冷饮为热，温饮为寒；口不渴而逼迫饮水为里寒；口干燥不欲饮为内有瘀血
6. **手足温度**——足冷为阳虚，冷至肘膝为里寒甚（四逆汤证）；手足热额凉为常
7. **掌额温度**——掌应温额应凉；掌凉额热为阳不入阴、阳浮于上
8. **问汗**——盗汗为表阳虚（桂枝加龙骨牡蛎）；无汗为表实（麻黄证）
9. **头身**——睡时身热为阳不入阴；阴阳律：男背阳胸阴、女胸阳背阴
10. **体力 / 生理期**——女子病必问月经，更年期妇女一日无两次大便以上者均有致七种癌之虞

#### 眼诊（倪师独创）

分心、肝、脾、肺、肾五区：
- **瞳孔对光反应迟钝** = 肾阳不足；瞳孔散大 = 肾阳大虚/受惊
- **心区扁平** = 心阳不足；心区偏大 = 阳不固阴；心区白 = 心血虚
- **肝区有破洞 / 褐点 / 圈纹** = 肝积毒或阴实；肝区损伤 = 肝血瘀
- **肺区有痰丝 / 白点 / 黄块中含黑点** = 肺寒痰湿或肺阴实（肿瘤）
- **脾区过小** = 脾虚
- **眼白区（心包区）痰多** = 心包痰湿
- **癫痫眼诊**：眼白上方三条鸡爪血丝连瞳孔（易治）；瞳孔上下均有（难治）；眼白下方连瞳孔（不治）
- 眼白上方三条血丝即[临床医案-倪海厦-0806.md](../原始资料/临床医案-倪海厦-0806.md) 中 Angelman syndrome 案鉴别要点

#### 耳诊、脊椎压诊、脉诊、触诊

- **耳诊**：探棒按压耳穴，五脏六腑相应点压痛即病；双耳肺点压痛→肺阴实；右耳肾区压痛→肾病
- **脊椎压诊**：T3 肺、T5 心包、T6 灵台（血癌）、T7 心、T9 肝、T11–T14 脾/肠；第六椎无压痛则血癌将愈
- **脉诊**：附骨脉（重按至骨仍有力）= 阴实/肿瘤；沉细数 = 少阴；洪大有力 = 阳明热；洪大中空（芤脉）= 血虚 / 心真脏脉；寸脉大于尺脉 = 阳气回复；一息八至九至 = 心阳欲竭
- **腹诊 / 触诊**：手掌干燥即摄护腺病（心火反逆入手掌）；痊愈标准为手掌温润+双脚极热；脐旁压痛 = 有宿便
- **特殊体征**：男无须刮 = 精子残渣不透；下利清谷 = 肾水泛大肠

#### 正常人六大标准（倪氏六准则）

出自[诊疗日志-倪海厦-0609.md](../原始资料/诊疗日志-倪海厦-0609.md)：

1. 正常的胃口，不超量也不没胃口
2. 每天早上一起床即上大号
3. 一天小便 5–7 次，淡清黄色
4. 一觉睡到天亮，无失眠无半夜醒
5. 常年头面冷而手足温热，睡时双脚须露在被外
6. 每天早起有阳反应（男勃起、女双乳敏感）

> 六大症状稳定 = 可停药；治疗期间六症渐复 = 治疗有效，反之无效。

### 五、独特临床见解

1. **阳不入阴（肿瘤成因）** — 局部温度低→营养冷水停积→压力升高→正常津液热能无力灌注→阴实越积越重→肿瘤。治癌五法：断瘤之源、以阳制阴、疏导攻坚、活血化瘀、常顾胃气。出自[诊疗日志-倪海厦-0812.md](../原始资料/诊疗日志-倪海厦-0812.md)。

2. **治肝治大肠** — 肝病必通大便，下利比便秘好；肝癌末期腹水大、大小便不出即不治。脐旁压痛+小便黄=有宿便，先攻宿使他症可分。出自[临床医案-倪海厦-0805.md](../原始资料/临床医案-倪海厦-0805.md)。

3. **渗利法 vs 通利法** — 腹水用轻剂渗利（茯苓泽泻芡实莲子薏仁陈皮通草），不用峻下通利——避免营养流失贫血补到癌细胞。西医抽水=通利法，水立刻回。出自[诊疗日志-倪海厦-0609.md](../原始资料/诊疗日志-倪海厦-0609.md)。

4. **治肺先治肝** — 见肺之病知肺传肝，治肺必治肝。常组：柴胡+郁金+黄芩+茜草+炙鳖甲。出自[临床医案-倪海厦-0805.md](../原始资料/临床医案-倪海厦-0805.md)。

5. **治肝泻南补北** — 泻心火（南）补肾水（北）使金不克木；戒烟方即以此理通关——清肝四宝（柴玉芩龙）+补肾阴（补骨脂）肾阳（泽泻巴戟天）+石膏知母缓胃口。出自[临床医案-倪海厦-0809.md](../原始资料/临床医案-倪海厦-0809.md)。

6. **心主七种癌** — 心功能不强→奶水不能完全下达子宫由月经出，或已无月经妇女一日无两次大便以上者，均有造成七种癌之可能：乳癌、脑瘤、血癌、肺癌、红斑性狼疮（Lupus）、肝癌、淋巴癌。出自[临床医案-倪海厦-0806.md](../原始资料/临床医案-倪海厦-0806.md)。

7. **奶水就是月经** — 女子奶水逆流入心=红斑性狼疮；入肺=肺癌；入肝=肝癌；入脑=脑瘤；入督脉=血癌。更年期妇女须保持大便畅通，宁可下利不可便秘。治女子病必问月经。出自[诊疗日志-倪海厦-0909.md](../原始资料/诊疗日志-倪海厦-0909.md)。

8. **阴阳律** — 男背阳胸阴，女胸阳背阴。肿瘤由胸至背（女）=阳入阴恶化；由胸至背（男）=阴出阳好转。

9. **节气治病（脏气法时）** — 心王夏（立夏治心），肺王秋，肝王春，肾王冬。治癌待病脏王时猛攻；5/5 立夏对阳气恢复极有利。出自[临床医案-倪海厦-0805.md](../原始资料/临床医案-倪海厦-0805.md) 255 案。

10. **热极生寒 / 真寒假热** — 雷诺氏手足趾冰冷+口含冰块=真寒假热；脉沉细标准少阴症。处方：麻黄附子细辛汤+当归四逆散合方。出自[临床医案-倪海厦-0809.md](../原始资料/临床医案-倪海厦-0809.md)。

11. **药简力专** — 重症越险药越简单；敢用附子（生附最高一两半）、硫磺、大黄、甘遂、藜芦。"治重病危症一定要药简力专，要敢用毒药峻剂。"出自[诊疗日志-倪海厦-0609.md](../原始资料/诊疗日志-倪海厦-0609.md)。

12. **生附与炮附共享** — 让体内如蒸气室般将阴实打开；生附石膏寒热药共享的特殊手法。出自[诊疗日志-倪海厦-0812.md](../原始资料/诊疗日志-倪海厦-0812.md)。

13. **汗多固表** — 上身汗多如水淋漓，重用炮附子至 8 钱固表使阳气不外泄往里走；生附子去里寒。出自[临床医案-倪海厦-0809.md](../原始资料/临床医案-倪海厦-0809.md)。

14. **胃口即病机** — 重症患者若胃口不佳时可用吴茱萸强化胃口；有恶心呕吐时便是使用人参补充津液的时机。出自[临床医案-倪海厦-0809.md](../原始资料/临床医案-倪海厦-0809.md)。

15. **寒凉鉴别** — 去里寒的药是生附子；生附子去心阳不足造成的里寒，当归四逆汤去血虚造成的里寒。出自[临床医案-倪海厦-0806.md](../原始资料/临床医案-倪海厦-0806.md)。

16. **心主五液** — 心功能回复时小便量增加、四肢逆冷去；寒热不去可能造成类风湿性关节炎。出自[临床医案-倪海厦-0806.md](../原始资料/临床医案-倪海厦-0806.md)。

### 六、跟诊心得核心教诲

摘自[诊疗日志-倪海厦-0812.md](../原始资料/诊疗日志-倪海厦-0812.md) 与[临床案例-人纪学生.md](../原始资料/临床案例-人纪学生.md) 收录的跟诊心得：

1. **知常达变** — 先掌握正常人六大标准（倪氏六准则），才能识别异常；治疗是否有效也以六准则判定。

2. **先辨阴阳** — "中医最高境界唯阴阳二字"；用动、静、时间、空间为变因观察人体：运动易汗否查心阳、不运动睡时汗否查表阳、晨起精神查肾阳、午前午后夜半身热查阴阳盛衰、身体分上下左右内外别阴阳。

3. **法象天道** — 心=太阳，肺=天，脾=地，肾=海，肝=木；心为君主之官、主汗、主喜、管睡眠、管四肢温度、管女人月经；无法治心脏病的医生无法治好癌症。

4. **忘记病名** — 不受西医病名迷惑，在症状间求答案。"如果诊断正确的话，开方治病又有何困难呢？"

5. **阴阳之道动而成圆** — "常人一定要动，不动就会生病。生病的人更需要动，不动就是死亡。"

6. **天人合一阴平阳秘** — 望气色与诊脉必考虑季节相生相克判病情凶险；节气脏气法时是治癌要诀。

7. **治心为要攻心为上** — 临床借幽默与强大气势让病人开心有信心；病人哭丧脸进来、咧开嘴出去，这一切都在治心。

8. **与癌共处并不难** — 未化疗放疗开刀切片者，治疗过程顺利且无痛楚；"If it is not broken, don't fix it!"；让病人把疾病交出来、把 happy 带回去。

9. **大框架思考** — 老师选桂枝龙骨牡蛎汤加减而非担法，让同学有"当头棒喝"之感——怕学生着相，须跳出病症框架看病人感受，先镇其心。

10. **第一击就击中** — 初诊全神贯注抓重点，让病人喝下第一碗汤剂即知已恢复大半，形成良性循环。

11. **中虚为明** — 临危不乱，辩证论治果断下药；毒甘蔗中毒案卢师兄一碗知。

12. **中西医学哲学差异** — 中医以"健康"为对象（放生），西医以"疾病"为对象（抓杀）；抗生素、疫苗、女性荷尔蒙、维他命、安眠药皆伤心致脚冷。

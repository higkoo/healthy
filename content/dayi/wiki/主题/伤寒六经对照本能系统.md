# 伤寒六经 vs 本能系统对照表

> **类型**: topic
> **创建时间**: 2026-07-21
> **来源**: [dayi-chuancheng-21qi-full](../../raw/dayi-chuancheng-21qi-full.md)、[A6-tu-jie-sheng-ming-ben-neng-xi-tong](../../raw/benneng-extra/A6-tu-jie-sheng-ming-ben-neng-xi-tong.md)

## 一句话说明

**把《伤寒论》1800 年前的"六经"翻译成现代人能懂的"功能高/低 + 外源/内源"两套坐标——不用背古文也能用伤寒方。**

## 核心对照表

郭老把《伤寒论》的"六经"重新翻译成"本能系统"的语言——
**太阳、阳明、少阳、太阴、少阴、厥阴**，其实就是**外源/内源 + 功能高/低**的六种组合：

| 六经 | 属于哪类病 | 病位 | 本能趋势 | 功能状态 | 代表方 |
|-----|----------|------|---------|---------|--------|
| **太阳病** | 外源性疾病 | 表/局部 | 向表排异 | 功能不低落 (+) | 麻黄汤、桂枝汤 |
| **阳明病** | 外源性疾病 | 里/局部 | 向上向下排异 | 功能不低落 (+) | 承气汤类 |
| **少阳病** | 内源性疾病 | 整体（三焦气机） | 自主调节 | 功能紊乱但**不低落** (+) | 小柴胡汤、柴胡桂枝汤 |
| **太阴病** | 外源性疾病 | 里/整体 | 自主排异 | 功能低落 (-) | 理中汤类 |
| **少阴病** | 外源性疾病 | 表/整体 | 自主排异 | 功能低落 (-) | 麻黄附子细辛汤 |
| **厥阴病** | 内源性疾病 | 整体（三焦气机） | 自主调节 | 功能紊乱**且低落** (-) | 乌梅丸 |

## 用一句话记

- **太、阳、少**（太阳、阳明、少阳）→ 功能还行（+）
- **太、少、厥**（太阴、少阴、厥阴）→ 功能低落（-）
- **太阳、阳明、太阴、少阴** → 是外源（要排异）
- **少阳、厥阴** → 是内源（要调节）

## 一张更直观的图

```
                        本能系统状态
                          │
              ┌───────────┼───────────┐
              │                       │
        功能不低落(+)             功能低落(-)
              │                       │
        ┌─────┴─────┐           ┌─────┴─────┐
        │           │           │           │
      外源        内源        外源        内源
        │           │           │           │
    太阳·阳明    少阳       太阴·少阴    厥阴
    （排异）   （调节）     （排异）    （调节）
                              │
                          功能极低落
                          （垂危）
```

## 闻道人（贾校长）的提醒

郭老把"功能 (+)"叫"正常值"，把"功能 (-)"叫"不正常"——**这说法不对**。
贾校长说：

> **这两种状态都是均势平衡、动态平衡的，都是正常的。
> 只有平衡被打破，疾病发生了才是不正常的。**

也就是说：老年人功能低落不等于"病了"，只是"功能值偏低"。
只有当他们出现具体病症时，才算生病。

## 为什么这张表重要？

它把《伤寒论》1800 年前的六经，**翻译成了现代人能懂的"功能高/低、外源/内源"两套坐标**，
让你**不用背《伤寒论》原文**，也能用伤寒方。

## 跟现代医学的对应

| 本能系统 | 现代医学近似说法 |
|---|---|
| 太阳病 | 感冒初期（身体还在抵抗） |
| 阳明病 | 肠胃型感染、高热 |
| 少阳病 | 植物神经紊乱、内分泌失调 |
| 太阴病 | 慢性消化不良、脾虚 |
| 少阴病 | 心衰、休克前期 |
| 厥阴病 | 慢性病晚期、多器官功能衰竭 |

（注：这是**功能层面的对应**，不是解剖学的对等）

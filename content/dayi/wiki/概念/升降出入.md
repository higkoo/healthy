# 升降出入

> **类型**: concept
> **创建时间**: 2026-07-20
> **来源**: [dayi-chuancheng-21qi-full](../../raw/dayi-chuancheng-21qi-full.md)

## 一句话说明

**人体能量的四种运动方向：升、降、出、入。升降出入顺畅 = 健康；卡住 = 生病。**

## 详细说说

### 这四个字哪来的？

《黄帝内经·六微旨大论》里说：
> **"升降出入，无器不有。非出入无以生长壮老已；非升降无以生长化收藏。"**

意思是：**所有有生命的东西都在做"升降出入"运动，没有例外。**

### 这四个字各是啥意思？

想象你身体里有一股"气"（能量、信息流），它得不断运动：

- **升** → 从下往上走（比如胃气上升到头脑，你就精神）
- **降** → 从上往下走（比如肺气下降到大肠，大便就通）
- **出** → 从里往外走（比如出汗、呼气、排精）
- **入** → 从外往里走（比如吃饭、吸气、喝水）

### 身体里的具体表现

| 升降出入 | 身体表现 |
|---|---|
| 升 | 胃气上升到头脑 → 精神 |
| 升 | 肾气上升到肺 → 呼吸深 |
| 降 | 肺气下降到大肠 → 大便通 |
| 降 | 心火下降到肾 → 睡得好 |
| 出 | 出汗、呼气、排便、射精、月经 |
| 入 | 吃饭、喝水、吸气 |

### 卡住的表现

- **气不升**：头晕、乏力、内脏下垂、脱肛
- **气不降**：呕吐、打嗝、咳喘、便秘
- **气不出**：发烧无汗、尿少、闭经
- **气不入**：没胃口、消瘦、营养不良

### 治病就是"恢复升降出入"

郭老说，**看病的关键就是看"升降出入"哪里卡住了**：

- 胃不升 → 用柴胡升提
- 肺不降 → 用厚朴下气
- 汗不出 → 用麻黄发汗
- 食不入 → 用山楂开胃

## 一个生活化的比喻

把城市想象成身体：
- 车（气）该上高架的上高架（升）
- 该下隧道的下隧道（降）
- 该进城的进城（入）
- 该出城的出城（出）

**一堵车（升降出入失常），全城瘫痪。**

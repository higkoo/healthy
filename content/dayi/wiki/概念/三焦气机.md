# 三焦气机

> **类型**: concept
> **创建时间**: 2026-07-20
> **来源**: [dayi-chuancheng-21qi-full](../../raw/dayi-chuancheng-21qi-full.md)

## 一句话说明

**古人叫"三焦"，郭老叫"三焦气机"——其实说的是身体里那股让能量升降出入的"总开关"。**

## 详细说说

### "三焦"是啥？

古人早就发现人体里有个东西叫"三焦"，但**1800 年来没人说得清它到底在哪、是啥**。

有人说它是"上、中、下三段"；
有人说它是"半表半里"；
有人说它是"非表非里"。

**莫衷一是。**

郭老在《伤寒六经求真》里给出了自己的答案：
> **三焦不是一个器官，是一个"功能系统"。**

### 它管啥？

郭老引用古人的话：
> "三焦为相火之用，分布命门之元气，主升降出入。"

翻译过来：
**三焦是把"命门"里的生命能量，分布到全身各处的"通道系统"。**

- **上焦** → 管纳入（吃饭、呼吸）
- **中焦** → 管生化（分泌、代谢、消化）
- **下焦** → 管排出（大小便、排汗、外分泌）

### 三焦 = 现代什么？

郭老自己说：
> **三焦气机 = 植物神经（自主神经）= 内分泌 = 生命共生性本能系统**

都是一回事。

| 不同时代的叫法 | 说的都是 |
|---|---|
| 张仲景 | 少阳 |
| 郭老早期 | 三焦气机 |
| 郭老后期 | 生命共生性本能系统 |
| 现代医学 | 植物神经紊乱、内分泌失调 |
| 俗称 | 亚健康 |

### 为啥它这么重要？

贾校长说：

> "三焦气机紊乱，就是植物神经（自主神经）紊乱，就是内分泌失调，
> 如果上升到本能理论，就是生命的共生性本能系统障碍。"

也就是说：
- **慢性病、亚健康**——根子大多在三焦气机失调
- **治疗慢性病**——核心就是调节三焦气机

### 一个生活化的比喻

把身体想象成一个家：
- **三焦**就是家里的"总电闸 + 电路系统"
- 它本身不是电灯、不是冰箱
- 但它管着电往哪里送、送多少
- **电闸跳了，全家电器都瘫痪**

所以你看，**三焦不是"具体的器官"，是"系统的功能"。**

### "气机"俩字

- **气**：能量，身体里流动的动力
- **机**：机枢、关键、原动力

**气机 = 能量的运转机制**

三焦气机 = 全身能量运转的总机制

## 跟我有啥关系？

如果你有这些状况：
- 一会儿冷一会儿热
- 胸口闷
- 没胃口
- 心烦
- 大便不通
- 睡不好

**很可能就是"三焦气机失调"**。
郭老治这类问题的代表方是**小柴胡汤**——
专门调节三焦气机。

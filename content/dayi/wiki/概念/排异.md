# 排异（排异反应、排异法）

> **类型**: concept
> **创建时间**: 2026-07-20
> **来源**: [dayi-chuancheng-21qi-full](../../raw/dayi-chuancheng-21qi-full.md)

## 一句话说明

**身体里有"不速之客"，本能系统就会想办法把它轰出去——这个"轰"的过程，叫排异；医生帮你"轰"得更顺，叫排异法。**

## 详细说说

### 排异是咋回事？

想象一下你家里进了苍蝇，你会怎么办？拿苍蝇拍打、开窗子往外轰。
身体的"排异"就是这个动作。

外面的病毒、细菌，里面的代谢垃圾、癌细胞、痰饮、坏死组织……
只要身体识别出"这不是自家人"，就会启动排异程序。

**排异的"出口"就这么几个**：

- **汗腺** → 出汗排异（治感冒常用）
- **大便** → 通下排异（治肠胃积滞）
- **小便** → 利尿排异（治水肿）
- **呕吐** → 涌吐排异（治胃里积食）
- **出血** → 破血排异（治瘀血肿块）
- **咳痰** → 肺的排异方式
- **月经、乳汁** → 女性特有的排异通道

### 发烧就是排异信号

**最常见、也最被误解的排异反应，就是发烧。**

发烧不是病，是身体在**烧热炉子、准备出汗赶病毒**。
你一吃退烧药，相当于一瓢冷水泼在炉子上——
病毒没赶走，反而被"逼"回身体深处，
就会变成肺炎、肾炎、风湿这些"后遗症"。

郭老反复说：
> **"流感发汗就好，退烧就出肺炎。"**

这就是为啥郭老治感冒用麻黄汤、桂枝汤——
**不是退烧，是帮身体出汗。**

### 排异法怎么用？

按排异通道选方：

| 排异通道 | 代表方剂 | 治什么 |
|---|---|---|
| 发汗 | 麻黄汤 | 风寒感冒 |
| 发汗 | 桂枝汤 | 中风（虚汗型感冒） |
| 发汗 | 透表汤 | 温病、麻疹、风疹 |
| 通大便 | 大承气汤 | 肠梗阻、积食 |
| 通大便 | 桃仁+大黄+丹皮 | 急性阑尾炎 |
| 利小便 | 五苓散 | 水肿、水逆 |
| 涌吐 | 瓜蒂散 | 胃里积食 |
| 破血 | 桂枝茯苓丸 | 瘀血、肌瘤 |
| 化痰 | 排异汤 | 肿瘤、囊肿 |

### 顺势利导：看身体想从哪儿排，就帮它从哪儿排

这是郭老看病的核心动作——**看势**。

举个例子：
- 感冒脉浮、怕冷、不出汗 → 身体想从汗排，但被堵住了 → 用麻黄汤帮它发汗
- 感冒脉浮、出汗、怕风 → 身体在汗排，但汗出不畅 → 用桂枝汤帮它调顺
- 肚子痛、大便不通 → 身体想从大便排 → 用承气汤帮它通

**医生的工作不是替身体做决定，是看懂身体想干啥，然后帮一把。**

## 排异 vs 对抗 —— 中西医思路的根本差别

| | 现代西医 | 郭老的中医 |
|---|---|---|
| 思路 | 对抗：杀菌、退烧、降压、降糖 | 顺应：帮身体排异、帮身体调节 |
| 发烧 | 用退烧药压制 | 用发汗药顺势 |
| 感染 | 用抗生素杀菌 | 帮身体把"敌人+垃圾"排出去 |
| 肿瘤 | 手术切除 | 帮身体把肿块"溶解排出" |

郭老最得意的一句话：
> **"我们不用抗毒药，也能治愈病毒性传染病。"**

## ⚠️ 警示：这些排异反应千万别压制

- **感冒发热**：不要立刻物理降温、用激素退烧
- **腹泻**：不要乱用止泻药（要先看是不是在排毒素）
- **咳嗽咳痰**：不要用强力镇咳药（痰要排出来）
- **出疹子**：不要用抗过敏药压回去

当然，**这不代表你可以不管不顾硬扛**。遇到急重症请立刻就医。
这里讲的是"理解身体的意图"，不是"啥都不做"。

## 跟我有啥关系？

下次身体出现"难受"的反应，先别急着压制，问自己一句：
**"这是身体在干啥？是不是在排毒？"**

如果是，就要**帮它排得更顺**，而不是**堵住它的路**。

## 📖 补充：发烧是排异（2026-07-21 ingest）

发烧是最典型、也最容易被误解的排异反应——身体"烧热炉子"是为了把体温升上去、把汗腺打开、把病毒随汗排出去。所以**真正的退烧是"排出去"，不是"压下去"**；一压，病毒留在体内，就可能变肺炎、肺感染。孩子、老人、积食三种发烧的辨别要点和安全提示，见 [发烧怎么办](../主题/发烧怎么办.md)。

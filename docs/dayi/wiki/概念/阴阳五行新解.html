阴阳五行新解（郭生白版）

阴阳五行新解（郭生白版）

类型: concept（理论） 创建时间: 2026-07-21 来源: 解密中医-郭生白 、本能系统医学-概论 、生命本能系统论

一句话说明

2005 年，郭老重新解读阴阳五行，从《内经·天元纪大论》“按图索骥"到《伤寒论》，破译出"五个生存本能”——这是他认为的"中医千年奥秘"的答案。

详细说说

注意：本页讲的是郭生白本人对阴阳五行的解读。 本 Wiki 另有一页 阴阳五行 ，讲的是贾校长在 21 期课程里的白话版（“阴阳=功能与物质，五行=五种关系”），两套说法有联系但不完全一样，请对照阅读。

郭老为什么要"新解"阴阳五行？

用他自己的话说（见 解密中医-郭生白 引言）：

  • 中医被说"博大精深"，但具体博大精深在哪儿，没人说得清
  • 孙中山、梁启超、鲁迅都说过：“中医能治病，但讲不清道理”
  • 中医自己说不出是怎么治好病的——很尴尬
  • 自唐宋以来流传的阴阳五行，已经不是中医原版的阴阳五行了
  • 医家的阴阳五行可能在汉代就失传了，《伤寒论》里有迹象但没明写

郭老要干的事：把中医原版阴阳五行重新挖出来。

破译过程：按图索骥

郭老的"图"是《内经·天元纪大论》里那段话：

“五行阴阳者，天地之道也。万物之纲纪，生杀之本始，变化之父母，神明之府也。”

他的"索骥"对象是《伤寒论》——张仲景的临床方法系统。

破译结果：从五行运动里推出五个生存本能——

五行运动（五种关系）           五大生存本能（五种能力）
─────────────────────────────────────────
阴阳离合而生杀            →   
相互依赖而生存            →   共生性本能
相互制约而均势            →   
相互变化而常新            →   应变性本能
终始嗣续而永恒            →   守个性本能
                          + 自主性本能（阴阳互根的动力源）
                          + 排异性本能（保护体内纯洁）

详见 五大生存本能 。

郭老的"阴阳"和"五行"到底是啥？

阴阳

  • 阴：有形的静物质（看得见的东西，比如血、肉、营养物质）
  • 阳：无形的动能（看不见的动力，比如心跳、出汗、思考）

关键概念：

  • 阴阳合德 = 生命开始（精子+卵子相遇）
  • 阴阳互根 = 生命过程（物质变能量、能量变物质，互相生发）
  • 阴阳离决 = 死亡（物质与动能分离）

五行

不是金木水火土！ 郭老明确说：五行是五种运动、五个系统——

  1. 阴阳离合而生杀（物种的发生与终结）
  2. 相互依赖而生存（谁也离不开谁）
  3. 相互制约而均势（互相管着，保持平衡）
  4. 相互变化而常新（一直在变，日日新）
  5. 终始嗣续而永恒（一代接一代，物种永续）

一句话概括（郭老原话）：

“相互依赖而生存，相互制约而均势，相互变化而常新，始终嗣续而永恒，阴阳离合而生杀。”

用生活例子串一遍

把你家厨房比作一个小宇宙：

  • 阴 = 锅、灶、米、菜（物质）
  • 阳 = 火、电、你的手艺（功能）

五行运动在厨房里：

  1. 阴阳离合而生杀：点火做饭=生，熄火洗完=杀
  2. 相互依赖而生存：没米做不出饭，没火也做不出饭
  3. 相互制约而均势：火太大糊锅，水太多煮成粥
  4. 相互变化而常新：每天菜色不一样
  5. 终始嗣续而永恒：今天吃完明天还得做

由这五行运动，又推出五个"生存本能"：

  • 你的身体会自己开火烧饭（自主性）
  • 火、水、菜配合着来（共生性）
  • 坏菜、馊水要倒掉（排异性）
  • 没煤气了会换电磁炉（应变性）
  • 你做的菜就是你做的菜，不会变成隔壁老王的（守个性）

这个"新解"厉害在哪儿？

郭老自己说的（解密中医-郭生白 ）：

  1. 解开了伤寒论六经之谜——六经不是经络，是"排异反应的不同阶段/部位"
  2. 解开了中医为什么不用抗毒药治传染病——因为用的是"利导排异法"
  3. 解开了中医为什么能一张方治糖尿病——因为用的是"自主性调节法"
  4. 解开了中医为什么超现代——因为它遵循生命本能的自然法则

郭老的自我评价：

“自宋成无己至民初陆渊雷先生，一千余年，多少医学家注解《伤寒论》，诠释的文字可多于《伤寒论》数百倍，但至今没有哪一位能说出六经是什么……直到把阴阳五行与生存本能说清楚，才把中医的十大难题一并解决。”

与贾校长版"阴阳五行"的对照

维度 郭老版（本页） 贾校长版（见 阴阳五行 ）
核心比喻 阴阳互根、五行运动 阴阳=功能/物质
五行解释 五种运动关系（生杀/依赖/制约/变化/永恒） 五种关系（离合/依赖/制约/变化/永恒）
重点 推出五大生存本能 讲功能是阳、物质是阴
用途 破译《伤寒论》的"道" 给学员讲懂基础概念

两套说法本质上是同一个体系的两种表述——郭老是源头，贾校长是 21 期课程里的通俗版。

⚠️ 重要提醒

  • “阴阳五行新解"是郭老个人学术主张，不是中医界共识
  • 他说"自唐宋以来的阴阳五行不是中医的阴阳五行”——这是非常激进的论断，主流中医界并不认可

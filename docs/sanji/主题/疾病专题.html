疾病专题

类型: topic 创建时间: 2026-07-21 最后更新: 2026-07-21 来源: 专题-乳癌篇-倪海厦.md 、专题-糖尿病篇-倪海厦.md 等

摘要

倪海厦对五种重大疾病各有专题论述：乳癌、艾滋病、糖尿病、肝癌、维他命危害。每个专题都以真实病例和中医病机为核心，批判西医治疗方式并给出经方治疗方案。乳癌的核心病机是「奶水逆流」——奶水即月经，心受损则奶水无法下达子宫而累积乳房，腐败成癌。糖尿病分三消（上消白虎人参汤、中消调胃承气汤、下消肾气丸），批判降糖药只降上半身血糖导致糖尿病足截肢。艾滋病被视为「Drug AIDS」——病人死于AZT等药物副作用而非病毒本身。

详情

一、乳癌专题

1.1 一个完整的治愈案例

倪师详细记录了Gloria May的乳癌治愈全程，分为三期：

第一期——恐惧怨恨期： 患者先因子宫水泡被西医做子宫卵巢全切除（化验正常，「根本只是一个水泡」），五年后发现乳癌。惊恐悲伤到极点，准备安排身后事。倪师门诊诊断：「女性患乳癌者，多起于情志郁结、心脏衰弱的病情」。倪师告诉患者：「药只能治三分病，其它七分还得靠自己」，患者起初失望，后领悟到必须自己从情志中走出来。

第二期——认识中医期： 患者在图书馆借到《走近中医》一书，开始理解中医辨证论治理论，对用中医治癌更有信心。服药期间主治疏肝理气、排便通畅（「如厕不须一分钟，我叫速便，而且排的非常痛快」），心情渐渐好转，积极返回工作岗位。

第三期——应证中医治疗期： 前三个月：服药后两小时癌块处有针刺般疼痛（「应证中医理论所说的药归经」）。五个月后：排出如牛奶一样的白色黏稠物（「应证了中医理论的排痰」），同时身上出现红疹。癌块时大时小但因有信心不再担心。一年多后：乳房恢复正常，已摸不到硬块。

倪师告诉她不需要西医切片检验——「任何西医的检验设备只会刺激癌细胞」；何时停药——「你自己会知道，因为治疗的结果感受在你的身体内，所以只有你自己最清楚」。

1.2 乳癌治疗的时机

「最快最有效的治疗乳癌时机是一开始发病时，妇女们注意一件事，当你发现乳房中有硬块时，中医不需要你去给西医鉴定是否是乳癌，因为西医的切片与挤压式的照射检查，只会让病情更坏……你只要发现乳房有硬块后就直接找中医治疗。」

倪师特别指出：台大医院一名医学院教授的太太（任台大护士长，最信西医），每年按要求体检，仍在52岁因乳癌死亡——「她可以不死的」。

二、糖尿病专题

2.1 三消论

  • 上消（口渴）：白虎人参汤
  • 中消（消谷善饥）：调胃承气汤类
  • 下消（小便多、阳不举）：肾气丸类

2.2 降糖药批判

按照物质不灭定律，降糖药只降上半身血糖，多余糖累积双脚→糖尿病足溃烂截肢（蒋经国案例）。胰岛素造成肝肾损伤→青光眼、肾衰竭。

2.3 倪师建议的防治方法

  • 不吃宵夜
  • 食不过饱七分饱
  • 禁抗生素（Cipro破坏胰脏升血糖）
  • 太极、走路、游泳
  • 找中医治疗
  • 足伤口：黄糖半斤温水浸泡半小时排出脓液，两三天愈合

三、艾滋病专题

3.1 Drug AIDS论

倪师核心观点：艾滋病本身无害，病人死于AZT/DDI/鸡尾酒疗法副作用。

  • 引用Peter H. Duesberg博士《Inventing The AIDS Virus》（1500+生化验证证明AIDS无害）
  • Magic Johnson不服药仍存活为证
  • Kaposi sarcoma实为西药AZT/DDI+毒品中毒引起，传统解毒法一夕消肿
  • 云南艾滋病8万+因吸毒人群多，海洛因后遗症被归为HIV
  • 结论：不吸毒就永远不会有AIDS

来源: 专题-愛滋病篇-倪海厦.md

四、维他命专题

4.1 核心论据

  1. 维他命C片：致动脉血管剥离+肾结石+支持乳癌
  2. 钙片：不但不补骨反而骨质流失+肾结石，需从小鱼干豆浆骨头汤摄取天然钙
  3. 维他命E：伤心脏，多种维他命滋养癌细胞
  4. 人工合成营养剂体积小：可被癌细胞病毒吸收
  5. 天然食物体积大碱性（蔗糖/柠檬/海盐）：癌细胞吃不下

4.2 五病所禁

筋病勿多食酸；骨病勿多食苦；肉病勿多食甘；气病勿多食辛；血病勿多食咸。

4.3 五味所伤

多食咸→脉凝泣变色；多食苦→皮槁毛拔；多食辛→筋急爪枯；多食酸→肉消皮厚唇开；多食甘→骨痛发落。

来源: 专题-維他命篇-倪海厦.md

五、肝癌专题

  • 热疗原理：无线电频波加热50℃烫死癌细胞，倪师理论认为热药阳药使水汽化→病毒癌细胞喜冷水→水变热渴死
  • 肝病治大肠：肝与大肠相通（无瓣膜血管），保持大便通畅是治肝关键
  • 渗利法vs通利法：腹水用渗利法（轻剂茯苓泽泻芡实莲子薏仁陈皮通草），不用通利法峻下——避免营养流失贫血
  • C型肝炎真相（倪师观点）：根本无C肝，实为维他命过量+感冒用抗生素→心脏受损免疫差→病毒入肝
  • 五味子：神农本草经上品，gomisin A强保肝成分，SGOT/SGPT保持正常

来源: 专题-肝癌篇-倪海厦.md

六、乳癌七类高危群

  1. 心情郁闷斤斤计较
  2. 喜吃西药维他命
  3. 心理压力大温柔婉约型
  4. 每年乳房体检妇女
  5. 服女性荷尔蒙+便秘
  6. 善嫉妒搬弄是非
  7. 一生辛劳为家庭事业忘我

七、14年存活标准

倪师指出：古代无西医治疗，乳癌可活14年；西医治疗后不超5年=死于治疗。三分之一乳腺癌无害不需治疗。切片=鸡蛋蛋黄切片后腐烂。发现硬块立刻找中医，不需切片。

什么是针灸？

针灸是中医两大治疗手段之一（另一个是中药）。针是用金属细针刺入穴位，灸是用燃烧的艾草在穴位上熏烤加热。针灸的使用比中药早了约一千年。

倪海厦把人纪 系列的第一部定为针灸——先搞清楚人体的经络穴位在哪里、怎么用，再学药就不会空中楼阁。

中医是物理医学，讲速度、温度、压力、时间。中医认为痛来自压力，疏解压力即可去痛，而非西医的杀菌。正因为是物理医学，可以"站在外边看到里面"。

教材以《针灸大成》（明·杨继洲）为主，参考《针灸甲乙经》《中医汇通医书五种》。针灸看扁鹊（祖师爷）、孙思邈、唐容川。

一、针灸使用时机与禁忌

不宜针刺的四种情况

  1. 气血大虚——重病、刚失血过多、病人完全没有元气
  2. 刚行完房事
  3. 激烈运动后大汗淋漓
  4. 喝醉酒后

原则：体力不够、元气很虚，不下针。

灸的禁忌

  • 热症不灸：感冒、发烧、痔疮等热性病不灸
  • 多汗多津液处不灸：头、脖、腋下、手掌、眼睛、阴部
  • 针而不灸、灸勿针：古书云"针灸并施，徒试患者炮烙刑"——同一穴位不要又针又灸

灸的原则

  • 痛灸到不痛：脓疱瘤之类，灸到好的皮肤
  • 不痛灸到痛：脂肪瘤之类，灸到患处
  • 灸单不灸双：3、5、7、9 壮（阳数）

二、灸法四种

灸用材料为艾草。艾草药性"往下找水、温热、纯阳"。古法找水源时，把艾草堆起来烧，方圆五里内冒烟的地方下面就有水。

1. 隔姜灸（最普遍）

  • 新鲜生姜切铜钱大小薄片，牙签戳几个洞
  • 艾草掌心搓成条状，揪一小段叫"一壮"
  • 姜片放穴位上，艾炷放姜片上点燃，姜片干了就换
  • 主治：几乎所有寒症；乳痈硬块初期隔姜灸软掉

2. 隔盐灸（神阙专用）

  • 海盐或青盐先炒热至温
  • 温热盐填入肚脐（神阙穴），艾炷放盐上烧
  • 主治：寒性腹泻（吃什么拉什么）、中风急救、霍乱下利、癃闭（膀胱炎、淋病、结石）
  • 热性腹泻（便很臭）不适合

3. 隔蒜灸（肺与咳嗽专用）

  • 一整颗大蒜切片，色白辛辣入肺
  • 放舌头上灸——舌为心表，舌往前顶心力加强
  • 主治：咳嗽等肺部问题

4. 米粒灸（鼻窦炎、鸡眼）

  • 艾绒揉成米粒大小，凡士林固定穴位上，香点燃
  • 主治：鼻窦炎（灸上星 7、9、11 壮）、鸡眼

特殊灸法

  • 结胸灸（肺积水）：黄连一钱 + 巴豆八颗去壳研粉，调和成饼放肚脐（神阙）上灸七壮，灸到肠鸣，病人上吐下泻即排水
  • 隔附子灸：脓疡无臭味的，用生附子或炮附子切片放伤口灸
  • 硫磺面饼灸：脓疡辛臭（如乳癌末期），白面粉、硫磺、大蒜等量做面饼，放伤口灸至少 21 壮，灸到腐烂肌肉排出

三、针灸急救五步法

遇人昏倒、中风、心脏病、溺水，不分原因按顺序：

  1. 针人中——嘴唇到鼻下 1/3 处，从下往上斜刺；无针则掐人中
  2. 手十宣放血——五指尖快速针刺挤血，可一次捉五指同刺
  3. 足十宣放血
  4. 针双脚涌泉——脚掌人字纹头，斜刺一寸半
  5. 百会放血——头顶正中放血，凉到眼睛能睁眼、凉到嘴巴能说话、凉到手脚能活动

来源：人纪-针灸.md （第一章·急救法）、针灸课笔记01-开篇.md

四、火罐的使用

使用时机

  1. 溃疡化脓——拔出脓头
  2. 瘀血——关节扭伤在痛处放血后拔罐
  3. 蚂蝗吸血排毒的替代

放血从暗黑色放到颜色变淡。

不用火罐的部位

腋下、乳房、头部、痔疮（痔疮刺破后直接火罐吸出脓血）。

五、穴位丈量与针刺通则

同身寸

  • 中指弯曲两横纹间为 1 寸
  • 三指并拢为 2 寸
  • 四指并拢为 3 寸
  • 胸腹部用均等法独立丈量

针刺深浅

  • 春夏下针浅，秋冬下针深
  • 病在肺刺皮毛，病在肾刺入骨边
  • 背薄似饼，腹深似井

入穴位置

穴位在：肌肉间、肌骨间、肌筋间、筋筋间、筋骨间。

成人与幼儿

  • 成人留针约 20 分钟
  • 14 岁以下幼儿单刺不留针
  • 出针时中指压皮肤，慢慢上提，快出针时停留 3–4 秒可不出血

常用针号

  • 32、28 号（硬）用于肌肉四肢
  • 一寸针 3210 用于手脚
  • 3230（32 号 3 寸针）用于透穴

七、五俞穴体系（井荥俞原经合）

十二正经都有井、荥、俞、经、合五个穴位，阳经多一个原穴。位置都在手肘以下或膝盖以下，安全性高。

  • 阴经五行排行：木火土金水
  • 阳经五行排行：金水木火土

五俞穴选用原则

俞穴 五行(阴经) 主治原则 说明
井穴 木 病在脏取井穴 手脚末梢，见效最快
荥穴 火 病变于色取荥穴 青肝、红心、黄脾、白肺、黑肾
俞穴 土 病在时间取俞穴 按时辰规律发作的病
经穴 金 病变于音取经穴 声音变了、嘴巴有味觉异常
合穴 水 病起于饮食取合穴 饮食不节、肠胃问题
原穴 — 五脏不平衡时用 阳经独有，安五脏

与四季关系

  • 井穴主冬、荥穴主春、俞穴主夏、经穴主长夏、合穴主秋
  • 长夏：节气转换时（春夏/夏秋/秋冬/冬春之交各两周）
  • 春夏养阳（气、力），秋冬养阴（有形）

五俞穴治疗法

  • 原络治疗：本经有病取原穴，配合表里经络穴
  • 俞募治疗：背部俞穴 + 腹部募穴，治内脏病基本套路
  • 会郄治疗：八会穴 + 郄穴，治急性痛症炎症
  • 子母补泻：经络上的痛（新病取子穴、久病取母穴）

九、奇经八脉

奇经八脉：任脉、督脉、冲脉、带脉、阴跷、阳跷、阴维、阳维。

十、十二正经要点

流注顺序与特点

肺 → 大肠 → 胃 → 脾 → 心 → 小肠 → 膀胱 → 肾 → 心包 → 三焦 → 胆 → 肝 → 回肺
  • 手三阴：肺（太阴）、心包（厥阴）、心（少阴）
  • 手三阳：大肠（阳明）、三焦（少阳）、小肠（太阳）
  • 足三阳：胃（阳明）、胆（少阳）、膀胱（太阳）
  • 足三阴：脾（太阴）、肝（厥阴）、肾（少阴）

脏腑官职（《黄帝内经》）

脏腑 官职 关键特性
肺 相傅之官 法象天幕；开窍在鼻；主皮毛
心 君主之官 心藏神（心中一滴血）；管奶水；与小肠表里
小肠 受盛之官 第二个心脏，火脏；余热到脚
脾 谏议之官 主少腹；主肌肉
膀胱 州都之官 水液储藏
肾 作强之官 伎巧出焉；主骨，其华在发，开窍在耳，主唾
肝 — 主筋；开窍在眼；1–3 点气血流注
心包 — 心的外卫，与心脏病态一致
三焦 — 行阳，分上中下焦
胆 — 胆汁喷射与心脏同步；11–1 点气血流注

常用要穴速查

  • 肺：中府（募）、尺泽（合水，子穴）、太渊（母穴/脉会）、列缺、孔最（郄穴，痔疮止血）、鱼际、少商（井）
  • 大肠：商阳（井）、二间（子穴）、三间、合谷（原穴/气穴/面口合谷收）、阳溪、偏历（络）、温溜（郄）、曲池（合土）、手三里

十一、补泻手法详解

基本补泻（留豆许）

  • 补：气到后往下推一点（“推豆许”），气在针下方
  • 泻：气到后往上提一点（“提豆许”），气往下走
  • 平补平泻：左转右转，病人感到酸麻胀（正常气至感觉，针旁一圈粉红色）

烧山火（九阳数，大补）

针下产生热感，强补。先引到气，分天人地三部，每部紧按慢提九次，共 27 次。找肉多的穴位做（足三里、三阴交、复溜），不要在井穴上做（会变成拷问）。胃寒在中脘或足三里做，肾寒在复溜或肾俞做。

透天凉（六阴数，大泻）

针下产生凉感，强泻。引气后从地部开始，紧提慢按，每部六次，共 18 次。治壮热、消渴（中消）、发烧实症（尺泽）。

阳中引阴 / 阴中引阳

  • 阳中引阴：先寒后热、酸多痛少 → 先补后泻
  • 阴中引阳：先热后寒、痛多酸少 → 先泻后补

留气法（治痞块、肿瘤）

远取穴，子宫/乳房硬块可下足三里、上巨虚、梁丘。行纯阳数九（紧按慢提九次），提豆许后停 5 分钟，再行阴数六（同一个穴位），留针。取针时不按针孔。

烧针（燔针）

治脂肪瘤、黑痣、化脓。三针法：第一针从上往下，第二针从前往后，第三针从后往前。麻油沾针，火斜烧三次，快速下针。良性才用（表面光滑），恶性硬如岩石不用。

报刺法（飞针引气）

治良性痞块。针从肿块上方下到底，左右捻后提到皮层，再向东南西北、东南、东北、西南、西北八方向各刺一针，回到原点捻针出针。

运气法（治四种痛症）

  1. 痛（实症） → 泻
  2. 痛多酸少 → 先泻后补
  3. 酸多痛少 → 先补后泻
  4. 酸（虚症/久病） → 补

提气法 / 中气法

调和中焦之气，提升气机。

飞经引气四法

手法 性质 动作要点
青龙摆尾 补 针在体内，左右摇摆针柄（摇铃状）九下
白虎摇头 泻 退方进圆，方形慢出、圆形进
苍龟探穴 补 一退三进，如乌龟入泥
赤凤迎源（赤凤摇头） 泻 退方进圆，进圆同时摇动针柄

做手法后一般不留针：补则按针孔，泻则不按。

龙虎交战 / 龙虎升降

  • 龙虎交战：捻转法，天人地三部各一补一泻（九六数）。止痛手法，针灸麻醉，治剧痛
  • 龙虎升降：插提法，三部各一次插提后让气循环。通关手法，治关节脱臼、气过不去

五脏交经法

青龙摆尾施于本穴（经属性相同的穴，如胆经属木、临泣也属木），使本经气血恢复，力量可达内脏。治痞块、内脏病。

通关交经 / 膈角交经

  • 通关交经：赤凤摇头，治关节断裂、脱臼后气过不去
  • 膈角交经：治心脏病先强金（肺），让病停止恶化

呼吸补泻法

患者吸气时拔针、呼气时进针为泻；反之为补。常用于环跳。

子午捣臼法

九入六出，治水胀。

下针八法

揣、爪、搓、弹、摇、扪、循、捻。

五门十变与天干歌

下本穴时不需青龙摆尾等手法，针下去左转右转引到气即可。

十四、针灸治症系列（按病分类）

头痛类

  • 全头痛：百会 + 涌泉（一导一引）
  • 偏头痛：对侧合谷 + 同侧太阳透率谷；恶心呕吐加中脘、内关、足三里、公孙
  • 前额痛/印堂痛：中脘（扎下立感酸，痛即去）
  • 眉棱骨痛：对侧阴陵泉下 1 寸肾关穴
  • 耳后痛：初痛扎天井，久痛扎中渚
  • 后脑痛/颈项痛：天应天柱；初痛束骨，久痛委中
  • 脑瘤：全身抖动眼上吊，下百会涌泉；绝骨压痛 + 命门压痛（实症长东西）；药物：全蝎 + 蜈蚣等量打粉装胶囊（息风止痉）

眼症

  • 红肿赤痛：攒竹、丝竹空、耳尖、太阳四穴点刺放血
  • 白内障/生翳膜：睛明、阳白、三间、养老、光明；肝俞、期门、太冲（肝开窍在眼）
  • 近视：同白内障针法

鼻症

  • 鼻息肉：迎香透内迎香（不透无效）
  • 鼻炎/鼻窦炎：合谷、通天、上星（米粒灸）、迎香；风池 + 对侧颧髎
  • 流鼻血：天府穴下针；不止则俞募治疗

耳症

  • 久鸣（虚）：补肾——肾俞、志室、京门、章门、复溜（青龙摆尾）
  • 初鸣（实）：然谷、外关、支沟
  • 耳门、听宫、听会：三穴轮刺法，分属三焦、小肠、胆经
  • 耳鸣化脓：同上加听宫

面部中风

  • 起因：大汗后受风（运动大汗吹冷气、骑车吹风）
  • 治疗：合谷（麻醉）→ 地仓透颊车 → 阳白透鱼腰 → 太阳透率谷 → 下关 → 廉泉；对侧针，歪左针右

  • 中风舌强不语：廉泉、承浆

舌症

  • 舌癌：同上加下针
  • 小儿不说话：同法刺舌

扁桃腺炎

  • 天应放血
  • 扁桃腺硬块：照海、列缺

甲状腺

肺症

  • 肺实（新咳剧咳）：尺泽泻

末期化脓：

  • 肝结石：同胆结石加肝俞
  • 脾肿大：脾俞、章门、痞根、三皇穴（阴陵泉、地机、三阴交）

  • 恶心呕吐：中脘下针 1 寸
  • 朝食暮吐：中脘、胃俞、脾俞、足三里
  • 腹膜炎：天应 + 水分 + 天枢 + 阑尾穴（奇穴）；慢性加脾俞、胃俞
  • 胃痛拒按（实）：梁丘 + 足三里泻
  • 胃痛喜按（虚）：足三里补

  • 吐血不止：中脘、内关（不用足三里）
  • 月经下血不止：隐白（留针 20 分钟，双侧更好）
  • 便血（大肠发炎）：中脘 + 温溜（会郄治疗）
  • 外伤出血：止血截法

  • 中消（吃百碗饭不能解饥）→ 中脘、足三里透天凉
  • 下消（阳不举）
  • 消渴穴（7–9 椎中间）压痛诊断
  • 条口透承山辅助诊治

  • 近取良性：烧针（三针法）或报刺法
  • 恶性（硬如岩石）：不用烧针报刺
  • 诊断压痛：痞根穴、消积穴（肿瘤）、三阴交（子宫卵巢肿瘤）
  • 子宫卵巢肿瘤：三阴交压痛拒按（实）→ 天应隔姜灸 + 隐白止血

  • 腹四针：中脘 + 关元 + 双侧天枢，治肠胃；小便淡白灸天枢，深黄下针
  • 寒实（无蠕动不痛）：灸天枢
  • 热实（腹痛便秘）：针天枢、支沟、照海
  • 瘀血腹痛：三阴交、血海、地机

盲肠炎/奔豚

  • 盲肠炎：阑尾穴（足三里下）压痛确诊，下阑尾穴 + 天枢 + 腹四针
  • 奔豚（气上冲心）：公孙、内关、巨阙、关元

血痢/痔疮/肠漏/大肠癌

  • 血痢：隐白、三阴交、中极、关元
  • 内痔（先血后便不痛）：长强、承山、孔最（灸止血）
  • 外痔（先便后血痛）：同上 + 天应放血拔火罐
  • 肠漏：白环俞、长强
  • 大肠癌：痞根、消积 + 腹四针；脚冰冷为特征

妇科

  • 痛经：三阴交 + 血海（痛时扎立止）；灸三阴交、中极、关元
  • 赤带（湿热味重）：下针——临泣、三阴交、血海、隐白、白环俞、水道、归来、曲骨
  • 白带（寒湿清淡）：同上 + 灸（寒湿需灸根治）
  • 月经提前（热）：下针
  • 月经延后（寒）：灸合谷、三阴交、中极、关元
  • 助孕：灸三阴交、关元、中极（问先生脚热否即可判断）
  • 绝孕：灸石门、阴交、绝孕穴
  • 子宫卵巢肿瘤：天应隔姜灸 + 隐白止血；三阴交压痛诊断
  • 怀孕忌针：腹部（肚脐以下）不针
  • 安胎：灸至阴、三阴交
  • 堕胎：至阴、三阴交、合谷（补合谷泻三阴交堕胎）
  • 胎位不正：针至阴（24 小时回正）
  • 无痛分娩：至阴、三阴交

中风

  • 中风八大穴：百会、肩髃、曲池、合谷、外关、环跳、风市、阳陵泉
  • 中风八大穴（半身瘫痪）：百会、通天、合谷透后溪、外关、曲池、肩髃、风市、阳陵泉透阴陵泉、绝骨透三阴交、天突、廉泉、风府、肾俞、京门、阴谷、委中
  • 预兆：手前三指麻（6 个月内中风）、足阳明胃经麻（2 周内）
  • 放血：十宣、百会、涌泉（中风放血急救）
  • 中风舌强不语：廉泉、承浆

水肿

  • 全身水肿：水分、水道、阴陵泉、地机、三阴交
  • 下半身水肿：肾——复溜、照海、阴谷
  • 脚踝水肿：心——内关、神门
  • 腹水：水分、水道、中脘、关元、三皇穴

关节/伤科

  • 膝五针：鹤顶 + 双膝眼（八字形）+ 阳陵泉透阴陵泉 + 阴陵泉，治一切膝关节疾病
  • 五穴四针：治关节炎
  • 踝扭伤：商丘透丘墟
  • 肩关节痛：条口透承山
  • 主妇手：养老透间使
  • 手指关节：三间透劳宫
  • 膝关节加强：膝关穴
  • 腰背：委中 + 阴谷
  • 关节脱臼：委中放血 + 龙虎升降
  • 风湿关节炎：膝五针 + 五穴四针
  • 脊椎痛/尾椎痛：承山放血（新伤青筋处）

皮肤病

  • 梅花针：拍击放血（皮肤癣、麻痹）
  • 皮肤癣：委中放血 + 合谷、曲池、三阴交、血海、筑宾
  • 脓疮：火罐放血
  • 青春痘（胃经）：合谷、曲池
  • 带状疱疹：头尾隔姜灸；或蜘蛛网（干棉花替代）盖疹点火；全蝎烤干装胶囊吃 8–9 粒

神志病

  • 癫痫：鸠尾、百会、申脉（昼发灸）、照海（夜发灸）
  • 失眠：神门、三阴交、安眠穴（奇穴）
  • 心神失眠：心俞、神门、巨阙
  • 眩晕：百会、风池、内关
  • 帕金森：百会、风池、外关、阳陵泉、绝骨
  • 梦游：十三鬼穴考虑
  • 狂症/发狂奔走：十三鬼穴按顺序下

十五、灵龟八法与子午流注

灵龟八法

  • 用八卦选八脉交会穴的开穴法，按日时干支计算
  • 特性：急救、急诊用；平衡人体
  • 8 穴分 4 组控制奇经八脉
  • 灵龟八法盘正面灵龟、反面子午流注
  • 开到 5 时：男扎照海、女扎内关（阴阳不同）
  • 什么病都可治，感冒发烧下完就好

子午流注

  • 按日时干支取十二经五俞穴的开穴法
  • 只看干，不看支（灵龟八法看干支）
  • 单针（一针），男左女右

八脉交会穴（公孙内关胃心胸）

  • 公孙 + 内关：胃、心、胸（胃出血、胃溃疡、气喘、久咳、肺结核、心脏肥大）
  • 后溪 + 申脉：督脉、阳跷
  • 足临泣 + 外关：带脉、阳维
  • 列缺 + 照海：任脉、阴跷

来源：人纪-针灸.md （第六章·灵龟八法）、针灸-目录.md

十六、回阳九针与十三鬼穴

回阳九针

歌诀：哑门劳宫三阴交，涌泉太溪中脘接，环跳三里合谷并，此是回阳九针穴。

九穴：哑门、劳宫、三阴交、涌泉、太溪、中脘、环跳、足三里、合谷。

  • 主治回光返照，用在最后一步
  • 病人断气但家属未到，下针后病人可醒约半小时交代遗言
  • 一般下到第六针病人即开始呼吸讲话
  • 穴道要下的准（足三里不可下成足二里）

十三鬼穴（孙思邈）

治癫痫、发狂奔走、中邪（男发女声、女发男声等不可理解事）。

  • 正取穴无效时使用
  • 必须按顺序下：人中 → 少商 → 隐白 → …（不能乱序）
  • 双穴：男先左后右，女先右先左
  • 下针时：男往左转，女往右转

来源：人纪-针灸.md （第六章·回阳九针与十三鬼穴）

特殊刺法

  • 膝五针：鹤顶 + 双膝眼 + 阳陵泉 + 阴陵泉
  • 腹四针：中脘 + 关元 + 双天枢，治肠胃
  • 五穴四针：治关节炎
  • 三皇穴：阴陵泉（天皇）+ 地机（地皇）+ 三阴交（人皇），治水肿
  • 担法：上下取穴（如肩井 + 足三里）
  • 截法：对应取穴
  • 对称治疗（缪刺法）：左病右治、右病左治（会阴痛扎百会、偏头痛对侧合谷）
  • 模拟法找阿是穴
  • 指针法：幼儿三关（风关、气关、命关）用指针
  • 五脏逼毒法：肾脏音"吹"——压肾弓身吹气，最后提肛无声一吹

  • 解穴（脚/手少府）：解针感、点穴
  • 治肝三穴：肝病
  • 绝孕穴：绝孕

十九、针灸疗程与禁忌

  • 针灸疗程中感冒：先治感冒再续疗
  • 针后禁忌：避免剧烈运动、房事、酒
  • 晕针：因姿势不好，刺激人中
  • 药不暝眩，疾弗瘳：药不产生暝眩，病不会好

  1. 正常胃口，不会超量吃也不会没胃口
  2. 每天早上起床第一件事上大号
  3. 一天小便 5–7 次，颜色淡清黄

  6. 每天早起有阳反应（男勃起、女双乳敏感）

摘要

《金匮要略》为张仲景《伤寒杂病论》的杂病部分，宋朝时被从《伤寒论》中分出独立成书。倪海厦将其定位为「分门别类的杂病论，从内科到妇科伤寒论的六经殿堂」——在六经辨证纯熟后进入金匮才能灵活运用。全书按病种分篇（脏腑经络先后病、痉湿暍、百合狐惑阴阳毒、疟、中风历节、血痹虚劳、肺痿肺痈、胸痹心痛、腹满寒疝宿食、痰饮咳嗽、消渴小便不利淋、黄疸、惊悸吐衄下血胸满瘀血、呕吐哕下利、疮痈肠痈浸淫、趺蹶手指臂肿转筋阴狐疝蛔虫、妇人妊娠产后杂病等），每篇包含多个经方方证。倪师讲解视频稿888K字符，在讲各篇时穿插文式易筋经12式教学，并将《千金要方》《肘后方》等后世方书附入。

详情

一、金匮上课前言（倪师总论）

1.1 金匮与伤寒的关系

「金匮是延续伤寒论而来的，属于杂病范畴之内，每个章节讨论不同的病，但是其辨证的基础还是以伤寒论为主的。」

「先研究伤寒论，等到所有经辨症条辨都能够很纯熟的运用之后再进入金匮时，就可以很迅速的进入这经方的殿堂。」

1.2 经方的历代传承

倪师在撰写时参考了大量历代经方大师著作：

  • 江阴曹颖甫《伤寒发微》《金匮发微》
  • 四川唐容川《中西汇通医书五种》
  • 陆九芝、傅青主、戴天章
  • 秦伯未、林直清校定《世补斋医书全集》
  • 江忍庵增批、林直清勘《徐灵胎医书全集》

倪师特别提到孙思邈：先写《千金要方》，数年后看到张仲景《伤寒杂病论》大为惊讶，于是将《千金要方》做了许多修正成为《千金翼方》——「足证历代名医都将经方视为唯一的最高指导原则，不知道使用经方的中医是不配被称为中医的」。

1.3 不立病名与同症同治

「中医学不立病名，是因为立病名会让疾病的定义变为更狭隘……中医学讲究同症同治，就是不受到病名的限制。」

「西医制造出许多吓人的病名，许多病人一听到自己得到某种病后当时就已经吓傻了……使得绝大多数的病人是因为西医药的过度治疗与心理惊吓而死亡的。」

1.4 金匮是人纪教学的终点

「这次的人纪教学，将在金匮讲授完毕后全部结束，而我的临床经验医案传承，将利用地纪来撰写，所有地纪中提出应用的处方都来自经方。」

倪师写于2007年5月20日，佛罗里达州自家书房。

三、倪师讲解金匮的独特方法

  1. 易筋经穿插教学：在各篇章之间穿插文式易筋经12式教学（第一篇附第一式、第四篇附第二式、第五篇附第三式）
  2. 《内经》脑中风急救：第五篇中风历节前附内经急救法
  3. 附方补充：将《千金要方》《肘后方》等后世方附入对应篇章（如炙甘草汤、獭肝散、消肿溃坚汤）
  4. 临床案例：穿插倪师临床医案讲解
  5. 八纲辨证表格：视频稿开头附「药方名-药材名-身体部位-病名-虚/实/表/里/阴/阳/寒/热」八纲分类表

四、补虚六弊

「疟证误补成痞；下痢误补成臌胀；咳嗽误补成肺痿；痰饮误补成肿；吐血泄精误补成劳；湿阻中焦误补成格。」

五、倪师临床核心观念

  1. 经方非寒不泄，非温不补：寒药（膏黄芩连）宣泄实热，温药（姜附椒）温补回阳
  2. 阴阳平衡：阳要封藏于内，阴在外固守。阳外越则危
  3. 治肝必治大肠：肝与大肠相通（无瓣膜血管），保持大便通畅是治肝关键
  4. 肾水凌心诊断：手掌发黑发青为肾毒入心，用黄连解毒
  5. 六经辨证是经方大门：娴熟伤寒论后进入金匮才能灵活运用
  6. 不立病名，同症同治：不受西医病名限制
